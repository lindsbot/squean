Welcome to the SQUEAN stack!

(Sequelize, Express, Angular Node + Passport)

Squean is a seed that gets you up and running fast with a functioning Node/Angular app with authentication (local strategy only).

Passport is prebundled and the local (non-OAuth) strategies are setup. Passport has awesome support for virtually any login strategy you need (140+). The maintainer frequently supports the project on Stack Overflow. Passport.js

This project was inspired by Linnovate's MEAN.io which is an awesome option if you are using MongoDB with Mongoose. Squean offers a similar jump-off point to get you rolling ASAP with your MySQL-Express-Angular-Node project.
